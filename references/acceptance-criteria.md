# Acceptance Criteria for AGENTS.md

## Required Checks
- `AGENTS.md` exists at repo root.
- Sections are in required order: Overview, Core Principles, Domain Philosophies (Master-Level), Product & Project Standards, 12 Golden Rules (Why / How / Check), Scope Boundaries, Permission Model, Execution Rules, Quality Bar, Decision & Accountability, Risks & Open Questions.
- Domain philosophies include Goal, Constraints, Evidence, Failure Cost, Tradeoffs, and Non-negotiables for each selected domain.
- 12 Golden Rules include Why/How/Check for each rule.
- Scope boundaries mention generated outputs and forbidden actions.
- Permission model states what needs approval.
- Decision and risk log locations are specified.
- Highest-risk failure mode is explicitly stated.
- Language is English and ASCII.
