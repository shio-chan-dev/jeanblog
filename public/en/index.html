Find First and Last Position of Element in Sorted Array

Find First and Last Position of Element in Sorted Array Summary Given a sorted array, return the first and last index of a target value, or [-1, -1] if not found. Use lower_bound and upper_bound. Approach l = lower_bound(target) r = upper_bound(target) - 1 If l is out of range or nums[l] != target, return [-1, -1]. Complexity Time: O(log n) Space: O(1) Python reference implementation from bisect import bisect_left, bisect_right def search_range(nums, target): l = bisect_left(nums, target) r = bisect_right(nums, target) - 1 if l >= len(nums) or nums[l] != target: return [-1, -1] return [l, r]

Find Target Indices After Sorting

Use bisect_left and bisect_right to get [l, r). Return list(range(l, r)). High-Value Sliding Window Applications: From Max Vowels to Real Monitoring

Sliding Window in Engineering: Max Vowels to Monitoring Summary Sliding window is a simple but powerful pattern. The LeetCode problem “Maximum Number of Vowels in a Substring of Given Length” is a clean entry point and maps to real monitoring windows. Approach (max vowels) Keep a window of length k, maintain a running count of vowels, and update the maximum. Complexity Time: O(n) Space: O(1) Python reference implementation def max_vowels(s, k): vowels = set("aeiou") cur = sum(1 for c in s[:k] if c in vowels) ans = cur for i in range(k, len(s)): if s[i-k] in vowels: cur -= 1 if s[i] in vowels: cur += 1 ans = max(ans, cur) return ans

Maximum Count of Positive and Negative Numbers

Maximum Count of Positive and Negative Numbers Summary Given a sorted array, return the maximum of the number of positive and negative values. Binary search gives the split points in O(log n). Approach Use bisect_left(nums, 0) to get count of negatives. Use bisect_right(nums, 0) to get first positive index. Positive count = n - right_zero. Complexity Time: O(log n) Space: O(1) Python reference implementation from bisect import bisect_left, bisect_right def maximum_count(nums): n = len(nums) neg = bisect_left(nums, 0) pos = n - bisect_right(nums, 0) return max(neg, pos)
