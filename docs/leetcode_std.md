A — Algorithm（题目与算法）
用通俗语言介绍题目
给 1~2 个基础示例（可用图/表）
C — Concepts（核心思想）
讲解本题核心算法思想

简述算法正确性与复杂度
给出替代方案对比（暴力、错误思路、为何当前最优）
S — Summary（总结）
总结本题的核心收获
给延伸阅读（类似题、教材章节、工程文档）
要求：
输出为 Markdown 格式
清晰、简洁、有教学感
可直接作为技术博客发布
